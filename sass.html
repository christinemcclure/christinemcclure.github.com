Compass

Ruby gem for process SCSS files into CSS

When you need to compress for production:

  1. Create production branch
    2. compass compile --output-style compressed --force <directory name>ad
  2. Verify that compresses CSS files were created
  3. Merge into master (Sass uses a different directory, so no worry about losing formatting)
